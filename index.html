EVA VLM
State Evaluator

Benchmarking vision-language model performance on astronaut state detection during Extravehicular Activity. Images from the NASA SUITS program are augmented and used to probe Gemini's accuracy and confidence calibration across three EVA operational phases in a zero-shot setting.

01 / Benchmark Design

Three-Phase EVA Classification

Each image is labeled with one of three EVA operational phases. The VLM predicts the phase and returns a confidence score. Augmented variants stress-test robustness to lighting, orientation, occlusion, and scale variation.

04 / Evaluation Model

Google Gemini

All evaluations use the Google Gemini API in a zero-shot setting — no fine-tuning or few-shot examples. The model receives raw augmented images and classifies the EVA phase from visual context alone.

Gemini 2.5 Flash — Recommended

Google's best price-performance model for vision tasks. Features native multimodal understanding, 1M token context, and built-in reasoning capabilities for batch evaluation at scale.
