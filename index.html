EVA VLM State Evaluator

Benchmarking vision-language model performance on astronaut EVA phase classification

A benchmark for evaluating vision-language model performance on astronaut state detection during Extravehicular Activity. Images from the NASA SUITS program are augmented across three EVA operational phases — Egress, Navigation, and LTV Repair — and used to probe model accuracy and confidence calibration under controlled distribution shift using Google Gemini in a zero-shot setting.

Benchmark Design

Three-Phase EVA Classification

Each image is labeled with one of three EVA operational phases. The VLM predicts the phase and returns a confidence score. Augmented variants stress-test robustness to lighting, orientation, occlusion, and scale variation.

Data Augmentation Pipeline

Evaluation Model

Google Gemini

All evaluations use the Google Gemini API in a zero-shot setting — no fine-tuning or few-shot examples. The model receives raw augmented images and must classify the EVA phase purely from visual context.

Gemini 2.5 Flash — Recommended

Google's best price-performance model for vision tasks. Features native multimodal understanding, 1M token context, and built-in reasoning. Well-suited for batch evaluation of image classification tasks at scale.
